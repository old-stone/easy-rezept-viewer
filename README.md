# Easy Rezept Viewer

## Overview

レセ電中の人向け、レセプト（請求書）ビューワー  
現段階では労災のみ対応

## Getting Started

### Prerequisites

- git 2.20.1
- node 11.14.0
- npm 6.9.0
- yarn 1.15.2

※開発時のバージョン

### Installing

```sh
git clone https://github.com/old-stone/easy-rezept-viewer.git
cd easy-rezept-viewer
yarn install
yarn start
```

http://localhost:3000/ にアクセス

## Running the tests

```sh
yarn test
```

## Deployment

```sh
yarn build
```
